# Campus Hub: The OS for Modern Education

Campus Hub is a comprehensive, AI-powered institutional management platform built to unify every department of an educational institution. From student registration to alumni management, Campus Hub provides a seamless, premium experience for administrators, teachers, parents, and students.

## 🌟 Key Features

- **AI-Powered Intelligence**: Automated document verification, assessment generation, and job description drafting.
- **Unified Student Management**: 360° student profiles, attendance, and disciplinary tracking.
- **Financial command Center**: Secure fee collection, automated invoicing (via Stripe), and expense tracking.
- **Operational Excellence**: HR management, library automation, inventory procurement, and transport/hostel oversight.
- **Community Engagement**: Dedicated portals for Parents, Teachers, and Students, plus a robust Communication Suite.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Explore Documentation**:
   Check the `docs/` folder for the full [Product Overview](docs/product_overview.md) and [Blueprint](docs/blueprint.md).

## 🛠️ Technology Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS with Radix UI (Shadcn)
- **AI Backend**: Genkit with Google Gemini
- **Payments**: Stripe
- **Hosting**: Firebase App Hosting
