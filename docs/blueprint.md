# **App Name**: Campus Hub

## Core Features:

- Dashboard: Interactive dashboard displaying key metrics (students, staff, fees, attendance) with quick links.
- General Settings: Settings panel for institute profile, fee structures, grading policies, and account management.
- Classes & Subjects: Centralized module for managing classes, subjects, and timetable assignments.
- Student Management: Student registration and profile management with features like ID card and admission letter generation.
- Accounts: Income and expense tracking with account statements.
- Fees Management: Fee invoice generation, collection, receipt generation, and reports. 
- Announcements: Admin and teacher announcements via system, student homeworks, exam generation.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and reliability.
- Background color: Very light blue (#E8EAF6). A desaturated version of the primary color for a clean look.
- Accent color: Purple (#7E57C2) for highlighting important actions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and short amounts of body text; 'Inter' (sans-serif) for body text.
- Use consistent and modern icons for navigation and module representation.
- Sidebar navigation for easy access to modules, content area for main functionality.
- Subtle transitions and animations for a smooth user experience.
